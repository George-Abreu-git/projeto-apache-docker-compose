Sobre o Projeto

Desafio: Aplicação HTML em Container Apache

Este projeto tem como objetivo demonstrar o processo completo de containerização de uma aplicação web estática utilizando Apache HTTP Server dentro de um container Docker.

A página foi construída com HTML5 e CSS3 puros, sem dependências externas de JavaScript, garantindo leveza e compatibilidade máxima. O servidor Apache serve os arquivos diretamente a partir do diretório /usr/local/apache2/htdocs, seguindo a configuração padrão da imagem oficial httpd.

O desafio consiste em empacotar todos os assets estáticos, configurar corretamente o Dockerfile e realizar o deploy do container expondo a porta 80, tornando a aplicação acessível via navegador em qualquer ambiente que possua o Docker instalado.
